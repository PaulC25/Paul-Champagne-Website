My Projects

FED Minutes NLP Analysis

This project takes in FED Minutes Transcripts between 2000 and 2023 and uses an NLP algorithm to determine the sentiment score of each meeting. Correlation and linear regression models are then used to determine the relationship between such meeting sentiments and popular indexes such as the S&P500, NASDAQ, DOW, and VIX.

Black-Sholes Options Calculator

This project uses the Black-Sholes model to calculate the call and put price, as well as the implied volatility of inputted tickers. The calculator also plots changes in contract prices based on volatility, time to maturity, and strike price of the security.

Download


World Bank Economic Data Panel

Using Haloviz panel, this project is a dashboard that uses the World Bank API. It allows the user to compare and visualize economic indicators such as Inflation and GDP growth between any countries of their choosing. It also allows the user (through another tab) to compare changes in economic indicators based on their input.
